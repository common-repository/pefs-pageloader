This plugin is a very simple tool that allows the user to show the render speed of the content between the header and footer tag. 


== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

Â 

Then activate the pageloader by using the tag** [pageloader] **where you would like to see the time it took for the page to be rendered.
